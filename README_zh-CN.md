# Opencode 开发环境开荒与在线

这个项目方对新电脑、虚拟机和容器，通过一条命令轻松搭建基础开发环境。使用 Opencode 作为 AI 协调器，后台定义了“开荒技能”来支持环境初始化。

## 功能

- 自动检测系统信息，包括操作系统和架构。  
- 由 Opencode 生成安装步骤并实时执行。  
- 支持 macOS 和 Windows 及 Docker / 租用机测试。  
- 最小可用的开发环境，包括 git、ssh和 node 或 python 运行环境。

## 目标

使新机开发环境搭建可重复、可分享和可视化，为开发者提供符合最佳实践的默认设置。

## MVP 范围

- 一条命令一键搭建开发环境  
- 确保包管理器（brew/winget）、git 和 ssh 及基础语言运行环境可用  
- 测试及服务器安全警告提示

## 路线图

- 里线碑 0: macOS 内部原型  
- 里线碑 1: 公开 MVP  
- 里线碑 2: 可扩展的技能来添加更多开发环境

## 协助

好奇心的开发者可以上手共同建筑更完善的“开荒技能”。
